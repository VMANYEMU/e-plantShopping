# e-plantShopping

A React-Redux shopping cart application for Paradise Nursery - an online plant store.

## Project Overview
Paradise Nursery is a plant shopping application built with React and Redux Toolkit. It allows users to browse houseplants grouped by category, add them to a shopping cart, and manage cart items.

## Features
- Browse plants grouped into categories (Air Purifying, Aromatic, Insect Repellent, Medicinal, Low Maintenance)
- Add plants to cart with dynamic cart icon counter
- Increase/decrease quantity or remove items from cart
- Real-time total cost calculation
- Built with React, Redux Toolkit, and Vite
